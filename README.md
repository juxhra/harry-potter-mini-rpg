# Harry Potter Adventure  
**Mini Project OOP Java - Console-Based Game**

![Harry Potter Adventure](https://img.icons8.com/color/96/000000/harry-potter.png)

> Sebuah game petualangan berbasis teks yang mengikuti kisah 6 tahun di Hogwarts, dengan sistem pertarungan strategis, eksplorasi lokasi, dan narasi dramatis — semua dalam konsol!

---

##  **Fitur Utama**

- **6 Tahun Petualangan**: Ikuti kisah dari Tahun Pertama hingga Pertarungan Akhir melawan Voldemort
- **Sistem Pertarungan Dinamis**:
  - Pilih mantra acak sesuai asrama
  - Gunakan ramuan penyembuh dengan sinopsis dramatis
  - Minta bantuan teman (Hermione, Ron, dll)
  - Gunakan item khusus (Jubah Tak Terlihat, Piala Api, dll)
- **Eksplorasi Lokasi**:
  - Pilih jalur masuk ke setiap lokasi (Menara Astronomi, Kamar Rahasia, dll)
  - Hadapi rintangan unik berdasarkan pilihanmu
- **Narasi Imersif**:
  - Sinopsis tahunan yang menjelaskan latar cerita
  - Ilustrasi ASCII spesifik untuk setiap lokasi
  - Dialog karakter yang hidup
- **Manajemen Sumber Daya**:
  - HP dibawa terus ke misi berikutnya (tidak otomatis penuh)
  - Ramuan terbatas (20 awal) — tidak otomatis pulih
  - Bisa menang **atau kalah** berdasarkan strategi
- **Karakter Kustom**:
  - Pilih jadi Harry Potter atau buat karakter sendiri
  - Pilih asrama: Gryffindor, Slytherin, Hufflepuff, Ravenclaw
  - Dapatkan teman sesuai asrama
